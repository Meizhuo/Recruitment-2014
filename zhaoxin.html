2014袂卓工作室招新信息

本次招新面向在校大一、大二本科生
总体要求：
1. 具有较好的耐心和较强的责任感;
2. 有良好的团队合作精神和较强的解决问题能力，具有良好的语言表达与沟通能力；
3. 对安卓开发、web开发、美工技术有一定的兴趣和了解;
4. 具备基础电脑操作技能和基本网络知识，创意丰富、见解独到者优先考虑;
5. 了解java、android、PHP、PS者优先考虑;

美工组：

简介：使用Photoshop或Fireworks等设计软件，将网页的视觉效果、排版等工作呈现出来
要求：
1. 对美有敏锐的洞察力；
2. 有创意和一点点完美主义；
3. 熟练使用PS或Fireworks;
4. 有设计经验，网页平面设计经验者优先考虑;

web开发小组：

简介：Web组以通过互联网丰富生活为主要目的，时刻与最新的趋势保持同步。浏览器是我们的世界，在这里我们可以尽情地毁灭和创造。主要方向：网站开发、Web App开发、浏览器插件开发、微信公众平台开发
小组要求：
1. 熟悉HTML、CSS等开发语言;
2. 喜欢探索新鲜有趣的技术;
3. 了解JavaScript、jQuery、PHP者优先考虑;
4. 有开发经验，重构过页面者优先考虑;

Android开发小组：

简介：Android组是基于Android系统以实现创意、探索技术为目的的小组，无论是在本地化功能型应用方向，还是基于大数据的客户端，都保持领先的技术水平和辉煌的战果。主要方向：手机游戏开发、功能型App开发、移动互联网应用及插件开发。
小组要求：
1. 热衷于互联网行业，有意向在该领域发展深造；
2. 有良好的团队合作精神和较强的解决问题能力；
3. 熟悉C/C++、C#、JavaScript、PHP、Python、 Java或其他现代程序开发语言中的任意一种，或者对其有浓厚兴趣;
4. 了解软件测试基本过程者优先考虑;
5. 有实际Android开发经验者优先考虑;
